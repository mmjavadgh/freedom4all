SingBoxUI is a graphical user interface application that allows users to connect to the SingBox service and manage their subscription. It provides an easy-to-use interface for starting and terminating the SingBox application and accessing the SingBox dashboard.

Usage

1. Download and unzip "SingBoxUI" to "Desktop".
2. right click on "SingboxUI.exe" and click on "Run as administrator".
3. in new window, click on "Yes".
4. Click the "CONNECT" button (red button) to start the SingBox application. (you may have click on "Allow" button in first time for new network adapter.)
5. Once connected, the IP address and location will be displayed.
6. Click the "DISCONNECT" button (green button) to stop the SingBox application.
* Click the "📃 OPEN SING-BOX DASHBOARD" button to open the SingBox dashboard. *


May help

* to "Run as administrator" every time automatically;
1. hold "Shift " button on keyboard and right click on "SingboxUI.exe"
2. click on "properties"
3. go to "Compatibility" tab
4. check "Run this program as an administrator"
5. click on "OK" button.

* To avoid error messages and file deletion by Windows Defender;
1. open "Windows Defender"
2. open "Virus & threat protection" (left menu)
3. in "Virus & threat protection settings" section, click on "Manage settings"
4. scroll down (at the end of the page), click on "Add or remove exclusions"
5. click on "Yes" (if "User Account Control window" shows up)
6. click on "Add an exclusion" button
7. click on "Folder"
8. in "Select Folder window" select "SingBoxUI" folder (in desktop).

*to create a shortcut in desktop;
1. hold "Shift " button on keyboard and right click on "SingboxUI.exe"
2. click on "Create shortcut"
3. now, cut the "SingboxUI - Shortcut" and paste to Desktop

Notes:

Add subscription;

1. Hamburger menu (☰)
2. group >> plus button (+) (top right)
3. type custom name in "Group Name"
4. Change "Group Type" from "Basic" to "Subscription"
5. paste "copied url from the "Mix Config Subscription" list to "Subscription Link" >> check button (✔)
6. back to home.

Esplit tunneling (apps);

1. Hamburger menu (☰)
2. Route >> plus button (+) (top right)
3. type custom name in "Route Name"
4. open "Application"
5. turn on apps you want to tunnel: "outbount = Proxy" (or don't want to tunnel: "outbount = Bypass")
6. scroll down and change "outbound" to "Proxy" or "Bypass" >> check button (✔)
7. back to home.

Exclude Routing (".ir" domains);

1. Hamburger menu (☰)
2. Route >> plus button (+) (top right)
3. type custom name in "Route Name"
4. open "domain"
5. type "domain:ir" and click on "OK" button
6. scroll down and change "outbound" to "Bypass" >> check button (✔)
7. back to home.

Exclude Routing (iran server but none ".ir" domains);

1. open "none"ir"domains" from here
2. copy all text inside this page into your clipboard
3. open NekoBox
4. Hamburger menu (☰)
5. Route >> plus button (+) (top right)
6. type custom name in "Route Name"
7. open "domain"
8. paste all text from your clipboard and click on "OK" button
9. scroll down and change "outbound" to "Bypass" >> check button (✔)
10. back to home.

Exclude Routing (Arvan Cloud IPs);

1. open "ips" from here
2. copy all text inside this page into your clipboard
3. open NekoBox
4. Hamburger menu (☰)
5. Route >> plus button (+) (top right)
6. type "Arvan Cloud" in "Route Name"
7. open "ip"
8. paste all text from your clipboard and click on "OK" button
9. scroll down and change "outbound" to "Bypass" >> check button (✔)
10. back to home.
